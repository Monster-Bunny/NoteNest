Commands

Music Commands

  • clearqueue: Clear the current player queue.
  • disconnect: Force Disconnect the bot from your voice channel.
  • join: Invite bot to your voice channel.
  • loop: Set loop mode to current song.
  • lyrics: Display lyrics for the current played song.
  • nowplaying: Show the current playing song.
  • pause: Pause the current played song.
  • play: Play your favorite song/s.
  • previous: Return to the previous song.
  • queue: Show the current player queue.
  • remove: Remove a song from the queue.
  • replay: Replay the current song.
  • resume: Resume the current paused song.
  • search: Search for your favorite song.
  • seek: Seek the current played song.
  • shuffle: Shuffle the current player queue.
  • skip: Skip the current played song.
  • skipto: Skip the current played song to a specific queue position.
  • stop: Stop or disconnect the player.
  • switch: Move the position of two songs.
  • volume: Set the volume of the current player.

Filter Commands

  • 8d: Set the current player filter to 8d.
  • disable: Clear the current player filters.
  • earrape: Set the current player filter to EarRape.
  • nightcore: Set the current player filter to Nightcore.
  • slowmode: Set the current player filter to Slowmode.
  • vaporwave: Set the current player filter to Vaporwave.

Utility Commands

  • 247: Toggle enable 24/7 in the voice channel.
  • autoplay: Autoplay random related song/s.
  • control: Display or hide the player control button. [Premium]
